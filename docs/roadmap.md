# StickerTrack Roadmap

## Completed foundation

- Localized collection catalog and personal tracking
- Manual code entry
- Camera and local-image OCR scanner
- Administrator catalog imports
- Administrator-controlled player enrichment and licensed images
- Optional external project support through a privacy-preserving link

## Future

### Live Sequential Scanner

Keep the camera open, detect a stable sticker, read its code, add it
automatically, wait for removal, and process the next sticker.

This feature is intentionally outside Phase 7. It must not be implemented
until stability detection, duplicate prevention, user confirmation, mobile
performance, and privacy behavior have separate approval.

Other deferred areas:

- Sticker-front recognition and image comparison
- Trading between users
- Public lists, friends, and social discovery
- First-party payments, subscriptions, and supporter benefits
# Phase 9

Production hardening is implemented: environment validation, restricted origins, security headers, structured logs, health endpoints, password reset provider boundary, account anonymization, production Docker and Compose, CI, legal pages, and operational documentation. Email verification enforcement and cookie-based authentication remain explicit pre-launch follow-ups. PWA work is deferred to Phase 10.
